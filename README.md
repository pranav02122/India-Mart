# India-Mart

A feature-rich B2C E-commerce web application inspired by Flipkart, designed to provide a seamless online shopping experience.

## Features

* **User Authentication:** Secure Login and Registration system for customers.
* **Product Catalog:** Browse and view products with a clean UI.
* **Shopping Cart:** Add, update, and remove items from the cart dynamically.
* **Order Management:** Place orders and simulate a checkout process.
* **Admin Panel:** Capabilities to manage products and view customer orders.

## Tech Stack

* **Backend:** Java, Servlets, JSP (JavaServer Pages)
* **Database:** MySQL (using JDBC for database connectivity)
* **Frontend:** HTML5, CSS3, Bootstrap, JavaScript
* **Server:** Apache Tomcat

## How to Run Locally

1. Clone this repository to your local machine.
2. Import the project into Eclipse IDE as a Dynamic Web Project.
3. Configure your Apache Tomcat Server in Eclipse.
4. Set up your MySQL database and update the connection details in your Java files (DAO layer).
5. Right-click on index.jsp -> Run As -> Run on Server.
